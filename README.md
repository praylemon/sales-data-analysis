# Sales Data Analysis (Python)

This mini project demonstrates basic data analysis using Python.

## Objectives
- Create a sample sales dataset
- Perform statistical analysis
- Visualize monthly sales trends

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

- Calculated average and total sales
